Pour l'utilisation du forum "pok_forum_*", vous devez utilisez ce template pour gérer les données des membres.<br /><br />
<b>ATTENTION :</b> ce template utilise une ressource sur le dossier "membre" ID 2.<br />
Il créer un article dans le dossier "profil_forum" dans le dossier racine du membre du nom de "profil_forum_index" qui contient dans "text" un tableau php de chaque informations du membre comme le signature, lieu, etc.
